# gama-academy-project-mentesa-api

Gama Academy - Full Stack (React+Node) - Projeto em Grupo

<br />

### 💎 Principais Tecnologias

- TypeScript
- Prisma
- PostgreSQL
- Express API Server

<br />

### 🚀 Local

```bash
$ npm install
$ npm prisma:migrate
$ npm prisma:seed
$ npm run dev
```

<br />

### 🐳 Docker

```bash
$ docker compose up -d
```
